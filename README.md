# 📊 HR ANALYTICS—Employee Attrition Dashboard

An in-depth analysis of employee attrition patterns across 50,000+ workforce records spanning multiple departments. This comprehensive dashboard leverages advanced analytics, interactive visualizations, and SQL-driven insights to identify attrition drivers, compensation trends, and strategic recommendations for workforce optimization.

---

## 📥 Dataset Used

📊 **HR Analytics Dataset - 50,000+ Employee Records**
- 🏢 Multi-departmental coverage (HR, IT, R&D, Sales, Finance, Operations)
- 👥 Complete employee demographics and career metrics
- 💰 Comprehensive compensation and benefits data
- 📅 3-year historical attrition and performance tracking

## 🔑 KEY INSIGHTS & FINDINGS

### 📉 **Attrition Overview**
- 📊 **Overall Attrition Rate**: 50.2% (11,000+ employees)
- 🔴 **Highest Risk Department**: R&D (51.2% attrition)
- 🟠 **Second Risk**: Sales (47.3% attrition)
- 🟢 **Lowest Risk**: Finance (35.8% attrition)

### 💰 **Compensation Insights**
- 💵 **Average Monthly Income**: $26,016
- 📊 **Salary Range**: $1,009 - $199,999
- 📈 **Income-Attrition Correlation**: Negative (higher pay = lower attrition)
- ⚖️ **Pay Equity Issues**: Significant gaps across departments

### 😊 **Employee Satisfaction**
- ⭐ **Overall Satisfaction Score**: 2.49/4 (Below Average)
- 📉 **Department Variance**: 1.8 - 3.2 range
- 🎯 **Top Satisfaction Driver**: Career Development
- ⚠️ **Critical Factor**: Work-Life Balance

### ⏰ **Overtime Impact**
- ⏳ **Employees on Overtime**: 49.7% (24,850+ employees)
- 📈 **Overtime Attrition Premium**: +32% higher than non-overtime
- 💸 **Annual Cost**: ~$5.2M in overtime compensation
- 🔴 **Burnout Risk**: High correlation with resignation
