# HR Analytics Project

## 📊 Overview
Analyzed 50,000 employee records across 6 departments using Excel, Power BI, Tableau, and SQL to uncover attrition drivers, workforce demographics, and compensation insights.

## 🛠 Tools Used
- Excel (Data Cleaning, Dashboard)
- Power BI (Interactive Visuals)
- Tableau (Exploratory Dashboards)
- SQL (KPI Queries)

## 🔑 Key Insights
- Attrition rate: 50.2% (highest in R&D at 51.2%)
- Avg. monthly income: $26,016
- Job satisfaction: 2.49/4
- Overtime (49.7%) strongly linked to attrition
